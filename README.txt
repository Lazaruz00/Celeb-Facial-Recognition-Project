-This project uses OpenCV and face_recognition libraries to carry out a facial recognition on 30 Hollywood celebrities.
-Each celebrity folder has at least 380-400 trainable pictures that are already cleaned and have had duplicates deleted.
-The following links are the libraries and pictures required for this project.
1) Opencv: https://drive.google.com/drive/folders/1DUnNHtZAUubGQezlp7tig20jHS6rv6GU?usp=sharing
2) face_recognition library: https://drive.google.com/drive/folders/1T1SjJ3R31uzcdIdJo1EfLnHO0BfyHsJm?usp=sharing
3) dlib: https://drive.google.com/drive/folders/1osOZhYZaQz_kvdTXKryByVb5v1ANppTi?usp=sharing
4) Images for training: https://drive.google.com/drive/folders/1kTZ6bLVjR5Fcyn5QFtBPmCZApVHUQ9ZF?usp=sharing
